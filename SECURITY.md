# Security Policy

## Supported Versions

The most recent tagged version of blockhosts-nft is supported.

| Version | Supported          |
| ------- | ------------------ |
| v0.5    | :white_check_mark: |

## Reporting a Vulnerability

If you believe there's a security vulnerability, please don't use the public forums. Please click the **Report a Vulnerability** button above.

- When the report is submitted, GitHub automatically adds the reporter of the vulnerability as a collaborator and as a credited user on the proposed advisory.
- Optionally, click **Start a temporary private fork** if you want to start to fix the issue.

### After the Report

- A member of the project will triage the reported vulnerability.
- The team will work to solve the issue.
- If more information is needed, a team member will open a discussion.
- If the report is determined to be a security risk, full credit will be given.
- If the report is determined to not be a security risk, the reasoning will be available to the reporter as a comment.
